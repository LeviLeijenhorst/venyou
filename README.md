# VENYOU website

Een simpele statische one-pager voor VENYOU.

## Aanpassen

- Teksten staan in `index.html`.
- Stijlen staan in `styles.css`.
- De site draait zonder build-stap en is geschikt voor GitHub Pages.

## Lokaal bekijken

Open `index.html` in de browser, of start een simpele server:

```bash
npm start
```
